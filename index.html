📌
'알수없음'이란?

스마트스토어에서 구매자 유입 경로를 추적하지 못했을 때 붙는 분류.
직접 URL 입력, 카카오톡 링크 공유, 일부 앱 내 브라우저 등에서 자주 발생한다.

Step 2 · 무엇을 알고 싶나

질문 구조화 — 큰 범위 → 작은 범위

🌐
네이버 관련 매출인가? 아니면 다른 채널인가?

스마트스토어에서 발생했으니 네이버 생태계 내 유입일 가능성이 높음.
그러나 메타·카카오 등 외부 광고가 스마트스토어로 연결되는 경우도 있음.

🔍
네이버 광고(검색광고·GFA)에서 온 것인가?

네이버 검색광고 → 스마트스토어 랜딩 구조는 클릭 → 전환이 직접 연결됨.
그러나 UTM/클릭ID가 없으면 알수없음으로 잡힐 수 있음.

📱
메타 광고에서 온 것인가?

메타 → 자사몰 연결은 pixel로 추적. 메타 → 스마트스토어 연결은 UTM 없으면 알수없음 처리됨.

🌿
오가닉(광고 없는) 유입인가?

네이버 쇼핑탭 자연 노출, 브랜드 검색, 직접 방문 등도 알수없음으로 잡힐 수 있음.

Step 3 · 어떻게 보나

가설별 가능성 검토

❌ 가능성 낮음
메타 광고 유입
메타 픽셀은 자사몰(cafe24)에만 연결되어 있음.
메타 → 스마트스토어 직접 연결 구조가 없으므로 해당 채널의 기여로 보기 어려움.
🔶 애매함
네이버 GFA (디스플레이 광고)
GFA는 클릭 후 스마트스토어 랜딩 가능.
UTM 세팅 여부에 따라 알수없음 처리될 수 있음.
GFA 비용 대비 전환 추적 필요.
✅ 가능성 높음
네이버 검색광고 (브랜드·일반 키워드)
검색광고 → 스마트스토어 랜딩이 일반적 구조.
클릭ID(nclickid)가 누락되면 유입 출처가 알수없음으로 잡힘.
브랜드 키워드 검색 후 구매 패턴과 시기적 상관관계 확인 필요.
🔶 일부 해당
오가닉 · 직접 유입
네이버 쇼핑탭 자연 노출, 찜/재구매, 카카오톡 링크 공유 등.
광고가 아닌 브랜드 인지도 기여분.
전체 알수없음 중 일부를 차지하나, 2,700만 원 전부를 설명하기엔 부족.
핵심 단서

가장 유력한 원인 — 네이버 검색광고 미매칭

🔑
왜 검색광고가 유력한가?

네이버 검색광고 클릭 → 스마트스토어 이동 시, 스마트스토어의 유입 분석 시스템이
nclickid를 제대로 파싱하지 못하면 '알수없음'으로 분류함.
특히 모바일 앱 내 브라우저, 카카오톡 인앱 등에서 파라미터 손실 잦음.

확인해야 할 것

1️⃣
알수없음 매출 증감 ↔ 검색광고 집행 일치 여부

검색광고를 많이 쓴 날에 알수없음 매출도 함께 올라가는지 날짜별로 비교.

2️⃣
알수없음 매출 vs 스마트스토어 전체 매출 비율

알수없음이 전체 스마트스토어 매출의 몇 %인지 월별 추이 확인.
비율이 일정하다면 구조적 미매칭, 급증한다면 특정 캠페인 효과일 가능성.

3️⃣
검색광고 전환 데이터와 스마트스토어 구매 수 비교

검색광고 리포트상 전환수와 실제 스마트스토어 구매수 갭이 크다면
그 차이가 알수없음으로 잡히는 것.

📋 현재 결론 (작업 가설)

🔴 메타 광고는 스마트스토어와 직접 연결 구조가 없으므로 주요 원인에서 제외.
🟡 GFA(네이버 디스플레이)는 UTM 추적 세팅 미비 시 알수없음 기여 가능, 단 금액 규모 대비 설명력 낮음.
🟢 가장 유력: 네이버 검색광고 클릭 후 스마트스토어 유입 시 클릭ID 파라미터 손실로 알수없음 분류.
🟡 오가닉(브랜드 검색·재구매·쇼핑탭 자연 노출) 일부도 포함되어 있을 가능성 있음.
